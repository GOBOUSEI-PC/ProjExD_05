# まりお

## 実行環境の必要条件
python >= 3.10
pygame >= 2.1

## ゲーム概要
マリオブラザーズを模したゲーム
操作：左右ボタンと「ｊ（ジャンプ）」
クリア条件：強制スクロール制で、最後のゴールにたどり着いたらクリア
参考サイト：https://magazine.techacademy.jp/magazine/24530


##ゲームの実装
###共通基本機能
ウィンドウと背景
### 担当追加機能
プレイヤー：操作キャラクターの動きに対するクラス
マップ　　：ゲームマップの作製
敵Mob 　　：敵キャラクターの動きに関するクラス
ゴール　　：クリア時の処理
ゲームオーバー：クリア失敗時の処理

基準の高さ(height)*(4/3)
